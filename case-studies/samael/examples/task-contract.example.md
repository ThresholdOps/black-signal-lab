# Task Contract Example

## Purpose

Define a bounded AI-assisted task before execution begins.

## Task

Create a public-safe outline for a governance case study about agent-assisted workflow review.

## Objective

Produce a concise Markdown outline that explains the workflow model, review checkpoints, and data boundary in language suitable for a public portfolio.

## In Scope

- public-safe problem framing
- governance-oriented workflow steps
- example artifact list
- validation and human-review notes
- explicit privacy and data-boundary statement

## Out of Scope

- private repository inspection
- real task logs
- internal URLs or private paths
- production configuration
- credentials, tokens, or secrets
- claims of uncontrolled autonomous execution

## Inputs

- public repository README
- public-safe task description
- synthetic example assumptions where needed

## Expected Artifacts

- draft case-study outline
- short validation note
- human-review handoff note

## Validation Gates

- scope matches the approved task
- no restricted data categories are present
- language describes AI-assisted bounded execution
- output is ready for human review, not automatic publication

## Human Review Required

A human reviewer must approve the final artifact before publication, commit, or external sharing.
